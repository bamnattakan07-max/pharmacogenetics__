# บทที่ 1 รหัสชีวิตกับยา
## "**DNA ทำงานอย่างไร ?"**

ลองนึกภาพตามสถานการณ์ที่อาจจะเกิดขึ้นจริงในชีวิตประจำวันของคุณ ยกตัวอย่าง คุณและเพื่อนสนิทมีอาการปวดศีรษะในคืนเดียวกัน พวกคุณทั้งคู่รับประทานยาแก้ปวดชนิดเดียวกัน

รับประทานยาในขนาดที่เท่ากัน แต่ผลลัพธ์กลับตรงกันข้าม ราวกับฟ้ากับเหว คุณรู้สึกดีขึ้นภายในครึ่งชั่วโมง ในขณะที่เพื่อนของคุณยังรู้สึกปวดศีรษะ และบอกว่ายาไม่ได้ผลเลย ในขณะที่อีกคนในบ้านรับประทานยาตัวเดียวกัน ในขนาดยาที่เท่ากันกลับรู้สึกคลื่นไส้จนแทบลุกไม่ขึ้น สถานการณ์นี้ไม่ใช่เรื่องบังเอิญ ไม่ใช่เรื่องของความแข็งแกร่งของร่างกายแต่ละคน และไม่ใช่เรื่องของจิตใจ แต่เป็นเรื่องของรหัสพันธุกรรมที่ซ่อนอยู่ภายในทุกเซลล์ของร่างกายเรา

ซึ่งยาที่ใช้กันอยู่ทั่วไปในปัจจุบันนั้น โดยส่วนใหญ่แล้วเป็นการผลิตขึ้นมาสำหรับผู้ป่วย "ทุกราย" เพื่อที่จะรักษาโรคใดโรคหนึ่งหรือหลายๆโรค แต่ความเป็นจริงคือ ไม่ใช่ผู้ป่วยทุกรายที่จะตอบสนองต่อยาที่ได้รับ อย่างตัวอย่างสถานการณ์ข้างต้น ที่บ่งบอกแล้วว่าลักษณะทางพันธุกรรมที่แตก
ต่างกันของมนุษย์เรานี้เอง ที่ส่งผลให้ผู้ป่วยแต่ละรายตอบสนองต่อยาได้แตกต่างกัน

และศาสตร์ที่ศึกษาความสัมพันธ์ระหว่างรหัสพันธุกรรมกับการตอบสนองต่อยานั้น มีชื่อว่า เภสัชพันธุศาสตร์(Pharmacogenetics) ซึ่งความรู้จากเภสัชพันธุศาสตร์ได้กลายมาเป็นรากฐานสำคัญสำหรับแนวคิด Precision Medicine หรือการแพทย์แม่นยำ ซึ่งมุ่งหวังให้ผู้ป่วยทุกคนได้รับ "ยาที่ใช่" ใน "ขนาดที่ใช่" และเหมาะสมกับพันธุกรรมของแต่ละคน ตั้งแต่ครั้งแรกที่เริ่มการรักษา แทนที่จะพึ่งกระบวนการลองยาผิดถูกที่เสียเวลาและอาจเป็นอันตรายต่อผู้ป่วย ทำให้ผลการรักษาดียิ่งขึ้นและมีผลข้างเคียงน้อยที่สุด

ซึ่งบทนี้จะพาผู้อ่านทุกท่านเดินทางสู่พื้นฐานที่สำคัญที่สุดในการทำความเข้าใจเภสัชพันธุศาสตร์ขั้นพื้นฐาน นั่นคือการทำความเข้าใจว่า DNA ซึ่งเป็นรหัสชีวิตของเราถ่ายทอดข้อมูลไปสู่การสร้างโปรตีนและเอนไซม์ที่ย่อยยาได้อย่างไร และความแตกต่างเพียงเล็กน้อยในรหัสนั้น สามารถเปลี่ยนแปลงผลของการรักษาได้อย่างมีนัยสำคัญ

### **1.1 Central Dogma: พิมพ์เขียวของชีวิต**

ก่อนที่เราจะเข้าใจว่า DNA นั้นเกี่ยวข้องกับยาอย่างไร เราจำเป็นต้องเข้าใจหลักการพื้นฐานที่สุดของชีววิทยาโมเลกุลเสียก่อน ซึ่งหลักการนั้นมีชื่อว่า **Central Dogma** ซึ่ง Francis Crick เสนอขึ้นในปี ค.ศ 1958 โดยอธิบายว่าข้อมูลพันธุกรรมเริ่มจาก DNA ไปสู่ RNA (Transcription) แล้วจึงกลายเป็นโปรตีนในท้ายที่สุด (Translation) ซึ่งการผลิตโปรตีนนี้เองที่จะกลายเป็นเอนไซม์สำคัญต่างๆภายในร่างกาย ที่มีหน้าที่สำคัญ อย่างการย่อยอาหาร การเผาผลาญยา เป็นต้น โดยกระบวนการนี้เปรียบได้กับสายการผลิตสามขั้นตอนที่ทำงานในทุกเซลล์ของร่างกายตลอดเวลา 

**DNA: ห้องสมุดพิมพ์เขียวขนาดใหญ่**

DNA หรือกรดดีออกซีไรโบนิวคลีอีก (Deoxyribonucleic acid) ประกอบด้วยตัวอักษรหน่วยย่อย 4 ชนิด ได้แก่ A(Adenine) T(Thymine) G(Guanine) และ C(Cytosine) เรียงต่อกันเป็นสายยาวในลักษณะเกลียวคู่ ซึ่งจีโนมของมนุษย์นั้นมีลำดับเบสทั้งสิ้น 3,200 ล้านคู่เบส หากลองนึกภาพการพิมพ์ลำดับเบสทั้งหมดลงในหนังสือ คงจะได้หนังสือเล่มหนาไม่ตํ่ากว่า 1,000 เล่ม แต่ร่างกายของมนุษย์เรานั้นสามารถอ่านและแปลรหัสทั้งหมดนั้นในเวลาเพียงไม่กี่นาที

โดย DNA ทำหน้าที่เป็น "ห้องสมุดพิมพ์เขียว" ที่บรรจุลำดับเบสทั้งหมดและถูกเก็บรักษาอย่างปลอดภัยในนิวเคลียสของเซลล์ ข้อมูลนี้จะไม่ถูกนำออกไปใช้โดยตรง แต่จะมีกระบวนการถ่ายสำเนาออกมาในรูปแบบ mRNA เพื่อนำไปสร้างโปรตีนแทน คล้ายกับห้องสมุดที่อนุญาตให้ถ่ายเอกสารออกไปใช้งานนอก
สถานที่ได้ แต่ต้นฉบับต้องเก็บไว้ในห้องสมุดตลอดเวลา

**mRNA: ใบสั่งทำของที่ส่งต่อข้อมูล**

เมื่อร่างกายต้องการสร้างโปรตีนชนิดใด เซลล์จะเริ่มกระบวนการที่เรียกว่า **Transcription** โดยเอนไซม์ RNA Polymerase จะอ่านลำดับเบสจาก DNA และถ่ายทอดออกมาเป็น **mRNA**(messenger RNA) ซึ่งเป็นโมเลกุลขนาดเล็กที่มีลำดับเบสพร้อมแปลเป็นโปรตีน และจะเดินทางออกจากนิวเคลียสไปยังไรโบโซมในไซโตพลาสซึม เปรียบได้กับใบสั่งทำของที่ส่งจากห้องออกแบบไปยังโรงงานผลิต

**โปรตีน: ผลิตภัณฑ์สำเร็จรูปที่ทำงานจริง**

ในขั้นตอนสุดท้ายที่เรียกว่า **Translation** ไรโบโซมจะอ่านลำดับรหัสบนmRNAและประกอบกรดอะมิโนเข้าด้วยกันทีละหน่วยจนได้โปรตีนที่มีรูปร่างและหน้าที่เฉพาะเจาะจง โปรตีนเหล่านี้ทำหน้าที่เป็น "เอนไซม์" ที่ขับเคลื่อนกระบวนการทางชีวเคมีทุกอย่างในร่างกาย รวมถึงการย่อยสลายยาที่เรากลืนเข้าไป

ความสำคัญของ Central Dogma ในเภสัชพันธุศาสตร์อยู่ที่ความจริงที่ว่า เอนไซม์ที่ทำหน้าที่ย่อยสลายยาในตับและอวัยวะต่างๆ ล้วนเป็นโปรตีนที่ถูกสร้างขึ้นจากกระบวนการนี้ทั้งสิ้น ดังนั้น หาก DNA ของคุณมีความแตกต่างจากคนอื่นในตำแหน่งที่สำคัญ โปรตีนหรือเอนไซม์ที่สร้างออกก็อาจทำงานได้ดีกว่า แย่กว่า หรือไม่ทำงานเลย ซึ่งส่งผลโดยตรงต่อร่างกายของคุณในการตอบสนองและจัดการกับยาอย่างไร
